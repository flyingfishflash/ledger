Entities:

  • Account
  • Transaction
  • Split
  • Commodity / Currency
  • Price
  • Institution

Account

  • Persisted as a Nested Set (Pre Order Tree Traversal)

Transaction

  • Collection of Splits
  • Minimum of two Splits

Split

  • Each Split is an amount credited or debited to an Account
  • The sum of the amounts of all splits associated with a transaction should be zero

Commodity

  • To store information about a commodity/security (stock)

Price

  • Price of a commodity or security at a given time
  • Updated via Yahoo Finance API
  • Updated manually

Institution

  • Institutions may be associated with one or more Accounts
